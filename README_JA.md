# Wingsearch 日本語版

Wingsearchを日本語で使いやすくするための改修版です。

## 主な変更

- 初期表示言語を日本語に変更
- 主要な検索画面の文言を日本語化
- 日本語カード名・能力文に加えて、英語名と学名でも検索可能
- 全角・半角、カタカナ・ひらがな、空白や中黒の違いを吸収する検索正規化
- 拡張セット名を日本語表示

## 起動方法

推奨環境はNode.js 14系です。

```bash
npm install
npm start
```

ブラウザで `http://localhost:4200/` を開いてください。

## 本番用ビルド

```bash
npm run build-local
```

成果物は `dist/wingsearch` に生成されます。

## 注意事項

- 元プロジェクトと同じGPL-3.0ライセンスに従います。
- 日本語翻訳データに未翻訳項目がある場合、その項目は英語で表示されます。
- 本アプリはWingspanの非公式ファンツールです。
